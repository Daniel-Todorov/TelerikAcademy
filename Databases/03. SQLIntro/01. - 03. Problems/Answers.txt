What is SQl?
- SQL is a declarative language for query and manipulation of relational data.

What is DML?
- DML is a part of the SQL language that deals with the CRUD operations on records in the database - Create, Read, Update, Delete.

What is DDL?
- DDL is another partof the SQL language that deals with creating, manipulating and deleting the database schemas as well as managing the permissions of the various users.

Recite the most important SQL commands.
- INSERT, SELECT, UPDATE, DELETE, CREATE, ALTER, DROP, GRANT, REVOKE.
